# Library Skills

This skill allows updating documentation and instructions for other libraries used in the project.

## Updating Skills

To synchronize skills with the installed dependencies, use the following commands:

### For Python (via uv):
```bash
uvx library-skills
```

### For Node.js / Frontend (via bun):
```bash
bunx library-skills
```
