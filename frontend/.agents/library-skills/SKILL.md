# Library Skills

Этот навык позволяет обновлять документацию и инструкции для других библиотек, используемых в проекте.

## Обновление навыков

Для синхронизации навыков с установленными зависимостями используйте следующие команды:

### Для Python (через uv):
```bash
uvx library-skills
